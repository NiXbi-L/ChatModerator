# Telegram Chat Moderator Bot: Руководство для новичков

Этот проект представляет собой Telegram-бота, который помогает модераторам чатов автоматически удалять рекламу и нецензурные выражения. Если вы не разбираетесь в программировании, но хотите запустить этого бота на сервере и подключить его к своему чату, это руководство поможет вам шаг за шагом.

---

## Что вам понадобится

1. **Telegram-аккаунт**: У вас должен быть аккаунт в Telegram.
2. **Сервер**: Это может быть виртуальный сервер (VPS) или хостинг, где будет работать бот. Примеры: [DigitalOcean](https://www.digitalocean.com/), [Hetzner](https://www.hetzner.com/), или любой другой VPS-провайдер.
3. **Токен бота**: Вы получите его при создании бота в Telegram.

---

## Шаг 1: Создайте бота в Telegram

1. Откройте Telegram и найдите бота **@BotFather**.
2. Начните диалог с **@BotFather** и отправьте команду `/newbot`.
3. Следуйте инструкциям:
   - Придумайте имя для вашего бота (например, `My Chat Moderator`).
   - Придумайте уникальное имя пользователя для бота (оно должно заканчиваться на `bot`, например, `MyChatModeratorBot`).
4. После создания бота, **@BotFather** выдаст вам **токен**. Сохраните его, он понадобится позже.

---

## Шаг 2: Подготовка сервера

1. **Купите сервер**: Выберите VPS у любого провайдера (например, DigitalOcean, Hetzner). Подойдет самый простой тариф (обычно достаточно 1 ГБ оперативной памяти).
2. **Подключитесь к серверу**:
   - Для Windows: Используйте программу [PuTTY](https://www.putty.org/).
   - Для macOS/Linux: Используйте терминал и команду `ssh`.
   - Введите IP-адрес сервера, логин (обычно `root`) и пароль, который вы получили от провайдера.

---

## Шаг 3: Установка необходимых программ на сервер

1. **Обновите систему**:
   Введите команду:
   ```bash
   sudo apt update && sudo apt upgrade -y
   ```

2. **Установите Python**:
   Введите команду:
   ```bash
   sudo apt install python3 python3-pip -y
   ```

3. **Установите Git** (если его нет):
   Введите команду:
   ```bash
   sudo apt install git -y
   ```

---

## Шаг 4: Загрузка и настройка бота

1. **Скачайте файлы бота**:
   Введите команду:
   ```bash
   git clone https://github.com/ваш-репозиторий/telegram-moderator-bot.git
   cd telegram-moderator-bot
   ```

2. **Настройте бота**:
   - Откройте файл `config.py` с помощью команды:
     ```bash
     nano config.py
     ```
   - Замените значение `token` на токен, который вы получили от **@BotFather**.
   - Сохраните изменения (в Nano: `Ctrl + X`, затем `Y`, затем `Enter`).

3. **Установите зависимости**:
   Введите команду:
   ```bash
   pip install -r requirements.txt
   ```

---

## Шаг 5: Запуск бота

1. **Запустите бота**:
   Введите команду:
   ```bash
   python main.py
   ```

2. **Проверьте работу бота**:
   - Откройте Telegram, найдите своего бота по имени и начните с ним диалог.
   - Отправьте команду `/start`, чтобы убедиться, что бот работает.

---

## Шаг 6: Добавьте бота в ваш чат

1. Откройте Telegram и перейдите в ваш чат.
2. Добавьте бота в чат как участника.
3. Сделайте бота администратором:
   - Откройте настройки чата.
   - Выберите "Администраторы".
   - Добавьте бота и дайте ему права на удаление сообщений.

---

## Шаг 7: Автозапуск бота на сервере

Чтобы бот работал даже после закрытия терминала, выполните следующие шаги:

1. **Установите `screen`**:
   Введите команду:
   ```bash
   sudo apt install screen -y
   ```

2. **Создайте новый экран**:
   Введите команду:
   ```bash
   screen -S bot
   ```

3. **Запустите бота**:
   Введите команду:
   ```bash
   python main.py
   ```

4. **Отключитесь от экрана**:
   Нажмите `Ctrl + A`, затем `D`. Теперь бот будет работать в фоновом режиме.

---

## Готово!
